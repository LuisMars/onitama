# Onitama Card Generator

Print-ready card generator for all 55 Onitama cards.

## Quick Start

1. Open `index.html` in your browser
2. Select card size
3. Choose card sets
4. Click **Generate Cards** and **Print**

## Card Sets (55 cards)

**Base Game (16):** Tiger, Dragon, Crab, Frog, Rabbit, Monkey, Mantis, Crane, Elephant, Goose, Rooster, Horse, Ox, Boar, Eel, Cobra

**Sensei's Path (16):** Bear, Fox, Dog, Giraffe, Iguana, Kirin, Mouse, Otter, Panda, Phoenix, Rat, Sable, Sea Snake, Tanuki, Turtle, Viper

**Way of the Wind (10):** Goat, Sheep, Bat, Eagle, Hawk, Lion, Octopus, Rhinoceros, Scorpion, Spider

**Promo (13):** Mejika, Kumo, Okija, Sasori, Lobster, Steer, Hornet, Centipede, Cat, Serow, Nessie, Butterfly, Moth

## Print Sizes

Recommended: **Poker/TCG (2.5" × 3.5")** - fits standard card sleeves

Other sizes: Bridge, Mini Euro, Standard Euro, Tarot, Square (70mm/88mm), Jumbo, Custom
